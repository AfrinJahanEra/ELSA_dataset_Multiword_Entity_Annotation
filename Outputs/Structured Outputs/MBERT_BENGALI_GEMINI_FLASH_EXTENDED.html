9. Key Observations

  • mBERT NER model works well for Bengali but produces subword splits in inference
  • NER performance dropped slightly on test set (expected domain gap)
  • Sentiment model is overconfident on validation set
  • Dataset size (3,000 samples) is relatively small for deep generalization
